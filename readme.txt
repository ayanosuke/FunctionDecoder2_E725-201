【アドレスの設定】
・3-127 基本アドレス
  CV29 2
  CV1  3〜127

・128-9999 拡張アドレス
  CV17 H
  CV18 L
  CV29 34

  例 アドレス1234
    16進数へ変換 0x04D2
    0xC000を加算 0xC4D2
    H と　Lに分けて、C4 D2 0xC4,0xD2
    10進数に変換 0xC4→196、0xD2→210
    CV17 196、CV18 210、CV29 34 を書き込みます。

回転量は16bit/Users/sugawamasashi/Documents/Arduino/DCC/Github/FunctionDecoder2_E725-201/FunctionDecoder2_E725-201/README.md
CV48 ステッピングモータカウントH
CV49 ステッピングモータカウントL : 20 くらいで1回転

実際の回転量は H*256 + L
20ピッチ動かす場合
20 * 20 = 400パルス
10進数で計算
400/256 = 1.56 ->1(H)  400-256 = 144(L)

16進数で考えると簡単
20 * 20 = 0x0190 -> 0x01,0x90 -> 10進数に変換　-> H:1 L:144
